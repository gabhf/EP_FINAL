# EP_FINAL 
Fernando Fincatti, Gabriel Huerta e Giovanna Campedelli - 1B 2019.1

Capture the Egg

O jogo funciona no estilo de Captura a bandera; no qual na versão digitalizada terá duas equipes de dois jogadores cada - sendo eles dragões. 
Cada equipe terá que capturar o ovo que está no catelo da equipe adversária e leva-lo de volta para sua base.
Durante a captura do ovo os dragões podem se atacar com suas bolas de fogo. 

Cronograma de evolução:
10/05: mapa com mascara + personagens se movendo sem ultrapassar paredes
17/05: mapinhas do castelo + egg 
24/05: ataque + hit points(volta para base se for acertado pelo inimigo) 
31/05: jogo pronto (se sobrar tempo features adicionais)
